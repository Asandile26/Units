The study of Units

I am going to present about units. We have two types of units, absolute units and relative unit.
absolute length units: are fixed and a length expressed in any of these appear exactly that size . they are recommended for printing lay out because most of them they can not be used on a screen, except pixel.Screens are different for almost everyone so it would be difficult to use exact cm or mm. When we talking about absolute lengths units we count, cm, mm, in, px, pt, pc

Relative units

Relative lengths specify length relative to another length property.Relative length units are relative to something else, perhaps the size of the parent element's font, or the size of the viewport. The benefit of using relative units is that with some careful planning you can make it so the size of text or other elements scales relative to everything else on the page
